# 📚 College Library Management System - LibraryX

A modern full-stack Library Management System built using **Flask**, **MySQL**, **HTML/CSS**, and **JavaScript** with an advanced dashboard UI, dynamic interactions, database procedures, triggers, and responsive design.

# 📦 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Library-Management-System.git
```

---

## 2️⃣ Open Project

```bash
cd Library-Management-System
```

---

## 3️⃣ Install Requirements

```bash
pip install flask mysql-connector-python
```

---

## 4️⃣ Setup MySQL

Start MySQL server.

Create database:

```sql
CREATE DATABASE library_db;
```

Import schema:

```bash
mysql -u root -p library_db < schema.sql
```

---

## 5️⃣ Configure Database

Open `library.py`

Update:

```python
host="localhost",
user="root",
password="YOUR_PASSWORD",
database="library_db"
```

---

## 6️⃣ Run Project

```bash
python library.py
```
